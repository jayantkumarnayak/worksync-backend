# WorkSync Backend 🚀

## Project Description
WorkSync is a backend application built using Spring Boot for user management and authentication.

## Tech Stack
- Java
- Spring Boot
- MySQL
- Spring Security
- JPA / Hibernate

## Features
- User Registration
- User Login
- Password Encryption (BCrypt)
- Validation
- Global Exception Handling
- Clean API Response

## API Endpoints

### Register
POST /api/users/register

### Login
POST /api/users/login

### Get Users
GET /api/users

## Security
- Password encrypted using BCrypt
- Sensitive data hidden using @JsonIgnore

## Testing
- Tested using Postman
